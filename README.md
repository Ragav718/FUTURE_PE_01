# AI Website Copy Generator for Local Businesses 🚀

Welcome to my project for generating high-converting website copy using AI! This repository contains structured prompts and AI-generated copy for a local business.

## 🏢 The Chosen Business

- **Name:** One Piece Cafe
- **Type:** Anime Themed Cafe & Restaurant
- **Location:** Anna Nagar, Chennai
- **Target Audience:** Anime fans, youth, families, and food enthusiasts seeking an experiential dining experience.

## 🎯 Objective
To design a prompt framework that generates conversion-focused website copy for a local cafe, eliminating generic AI text and focusing on clear value propositions, specific services, and strong Call-to-Action (CTA) sections.

## 📂 Repository Structure

- `prompts.md`: Contains the structured, repeatable prompt system (System, Homepage, Services, CTA).
- `generated_copy.md`: The final output generated by the AI, ready to be plugged into an AI Website Builder (like Lovable or Framer).

## 🧠 Prompt Logic

The prompts were designed to achieve the following:
1. **System Prompt**: Sets the AI as an expert copywriter, defining the exact tone (fun, adventurous, anime-inspired but accessible).
2. **Homepage Prompt**: Focuses on the "Hero" section. The goal is to grab attention instantly with a strong H1 (value proposition) and an inviting intro.
3. **Services Prompt**: Structures the offerings with a clear "What's Included" and "Why Choose Us" to answer the customer's implicit question: *What's in it for me?*
4. **CTA Prompt**: Combines urgency with the cafe's theme to encourage actual footfall and table bookings, ensuring the copy drives business results.

## 🛠️ Tools Used
- **AI Tool:** Google Gemini 3.1 Pro (used to design the prompts and generate the copy)
- **Website Builder:** Lovable (https://lovable.dev)
