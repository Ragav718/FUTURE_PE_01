# AI Website Copy Generator Prompts

These structured prompts are designed to generate high-conversion website copy for **One Piece Cafe**, located in Anna Nagar, Chennai. 

## 1. System Prompt
**Use this prompt to set the AI's persona before asking for specific copy.**

```text
Act as an expert, high-converting copywriter for local businesses. Your client is "One Piece Cafe", a themed cafe located in Anna Nagar, Chennai. The cafe is entirely themed around the popular anime "One Piece". 

Your goal is to write website copy that is:
- Highly engaging and adventurous, capturing the spirit of the anime.
- Clear and benefit-driven, focusing on the quality of food, ambiance, and the unique experience.
- Persuasive, using strong calls-to-action (CTAs) that encourage footfall and table reservations.
- Friendly, immersive, and accessible to both die-hard anime fans and casual cafe-goers.

Do not use generic fluff. Use nautical, pirate, and anime-inspired language subtly without being overly cheesy. Keep sentences punchy and scannable for web reading.
```

## 2. Homepage Copy Prompt
**Use this prompt to generate the hero section and introduction.**

```text
Write the homepage copy for One Piece Cafe. Include the following sections:
1. **Headline (H1):** A strong, clear value proposition (max 8 words). It must be catchy and reference the adventure/cafe theme.
2. **Sub-headline (H2):** Explain exactly who the cafe is for and the main benefit (e.g., great food, immersive anime experience in Anna Nagar).
3. **Hero CTA:** A primary button text.
4. **Short Intro Section:** 2-3 sentences welcoming the "crew" (customers) to the cafe, highlighting the ambiance, the delicious food, and the location in Anna Nagar.
```

## 3. Services / Menu Highlights Prompt
**Use this prompt to generate the offerings section.**

```text
Write the 'Our Offerings' (Services/Menu) section for the website. Create 3 distinct offerings/features of the cafe:
1. **Themed Food & Beverages:** (e.g., Devil Fruit mocktails, Pirate King burgers). Describe what's included and why it's delicious.
2. **Immersive Anime Ambiance:** Describe the decor, photo booths, manga reading corners, or anime screenings.
3. **Private Events & Nakama Gatherings:** Offer the space for birthdays, fan meetups, or private parties.

For each offering, provide:
- A catchy Service Name
- A short description (2 sentences) of what's included
- Why customers should choose this (the unique selling proposition)
```

## 4. Call-to-Action (CTA) Prompt
**Use this prompt to generate the final contact and urgency sections.**

```text
Write two persuasive Call-to-Action (CTA) sections for the bottom of the website:
1. **Booking/Reservation CTA:** Encourage visitors to book a table for their crew. Make it sound exciting (e.g., "Set Sail with Us"). Include dummy contact details and a location-based urgency prompt (e.g., mentioning Anna Nagar).
2. **Trust-Building / Social Proof CTA:** A short prompt encouraging them to follow the cafe on Instagram for daily adventures and secret menu items. 
Keep the tone friendly, urgent, and perfectly aligned with the pirate/adventure theme.
```
